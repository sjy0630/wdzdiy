# 电路分析

> 状态：🚧 整理中

## 推荐教材

- **《电路》** — 邱关源 第 5 版，国内最主流
- **《Fundamentals of Electric Circuits》** — Sadiku，配大量例题与仿真
- **《Electric Circuits》** — Nilsson

## 学习要点

- [ ] KCL / KVL、节点法、网孔法
- [ ] 戴维南/诺顿等效
- [ ] 一阶/二阶电路暂态分析
- [ ] 正弦稳态、相量法
- [ ] 频率响应、谐振电路
- [ ] 二端口网络

## 仿真工具

- **LTspice**（免费，模拟电路仿真首选）
- **Multisim**（教学常用）
- **PySpice**（Python 调用 ngspice，自动化分析方便）
