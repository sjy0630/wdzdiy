# 数字电子技术

> 状态：🚧 整理中

## 推荐教材

- **《Digital Design》** — Mano，经典教材
- **《数字电子技术基础》** — 阎石，国内主流
- **《CMOS VLSI Design》** — Weste / Harris（进阶到 IC 视角）

## 学习要点

- [ ] 数制与编码
- [ ] 布尔代数、卡诺图化简
- [ ] 组合逻辑：加法器、译码器、多路选择器
- [ ] 时序逻辑：锁存器、触发器、状态机
- [ ] 存储器：SRAM / DRAM / ROM 原理

## 与后续课程的衔接

数电学完应自然过渡到 [Verilog](../digital_ic/verilog.md) 与 [数字 IC 设计流程](../digital_ic/design_flow.md)。
