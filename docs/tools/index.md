# 工具与环境

工程师的效率在很大程度上由工具熟练度决定。微电子方向的工具栈和软件开发不完全一样，但有大量重叠。

## 本章内容

- [Linux](linux.md) — 大部分 EDA 工具运行环境
- [Git](git.md) — 版本管理
- [Vim](vim.md) — 服务器/远程开发必备
- [LaTeX](latex.md) — 论文 / 实验报告排版
- [Tcl / Make 脚本](scripting.md) — EDA 工具控制脚本

## 学习建议

!!! tip "效率投资"
    这些工具看起来 "和专业课无关"，但**省下的时间会以指数级回报**。
    建议先把 Linux + Git + Vim 三件套花一两周练熟，受益终身。
