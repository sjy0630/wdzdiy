# Vim

> 状态：🚧 整理中

## 为什么要学

EDA 工具运行在远程 Linux 服务器上，**没有 GUI 编辑器**时 Vim 是最快的选择。
即便是日常开发，Vim 模式编辑也能显著提升效率。

## 推荐资源

- `vimtutor`（终端直接输入即可）
- **《Practical Vim》** — Drew Neil
- vim-galore — <https://github.com/mhinz/vim-galore>

## 上手要点

- [ ] 三种模式：Normal / Insert / Visual
- [ ] hjkl 移动 + w/b/e + f/t 跳转
- [ ] 组合命令：`d`/`c`/`y` + 动作（如 `daw`、`ci"`）
- [ ] 寄存器、宏
- [ ] `.vimrc` 基本配置

## 推荐插件

- **NeoVim + LazyVim** — 一站式现代配置
- **vim-verilog** / **vim-systemverilog** — Verilog 高亮与缩进
