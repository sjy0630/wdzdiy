# 贡献指南

非常欢迎你来一起完善这份指南！

## 三种贡献方式

### 1. 提 Issue（最轻量）

在 [GitHub Issues](https://github.com/sjy0630/wdzdiy/issues) 反馈：

- 内容错误 / 链接失效
- 想看但还没有的内容
- 推荐资源补充

### 2. 直接编辑（推荐）

每一页右上角有 :material-pencil: 图标，点击即可跳转到 GitHub 在线编辑该 md 文件，
完成后 Fork + Pull Request，整个过程在浏览器内即可完成。

### 3. 本地开发

```bash
git clone https://github.com/sjy0630/wdzdiy.git
cd wdzdiy
pip install mkdocs-material
mkdocs serve   # 浏览器打开 http://127.0.0.1:8000
```

## 写作风格建议

- 每一页**先列教材 / 网课**，再列学习要点
- 学习要点用 task list 形式（`- [ ]`）方便读者打勾
- 公式用 `\(...\)`（行内）/ `\[...\]`（独立）写
- 推荐资源时附上一句"为什么推荐"
- 不堆砌内容，宁可少而精
