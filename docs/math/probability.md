## 学习要点

- [ ] 随机变量与分布
- [ ] 大数定律与中心极限定理
- [ ] 数理统计：参数估计、假设检验
- [ ] 随机过程基础（平稳性、自相关、功率谱）

## 与微电子的衔接

- **噪声分析**：热噪声、闪烁噪声、相位噪声都是随机过程
- **工艺偏差与良率**：Monte Carlo / Corner 仿真本质是统计采样
- **数据转换器**：ADC/DAC 的 INL / DNL / SNDR 都是统计指标
