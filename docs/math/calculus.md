## 学习要点

- [ ] 极限、连续性的 ε-δ 定义
- [ ] 一元微分与积分（重点：换元、分部、级数）
- [ ] 多元微积分（偏导、全微分、重积分、曲线/曲面积分）
- [ ] 常微分方程（一阶、二阶常系数线性方程）
- [ ] 级数（Taylor / Fourier 级数 — 后续信号与系统的入口）

## 与微电子的衔接

例如电容充电过程：

\[
v_C(t) = V_S \left(1 - e^{-t/RC}\right)
\]

这里的微分方程 \(RC \dfrac{dv_C}{dt} + v_C = V_S\) 就是 RC 电路最基础的描述。
