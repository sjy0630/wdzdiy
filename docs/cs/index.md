# 编程入门

微电子专业的同学不需要把自己练成纯软工，但**至少要熟练 1-2 门编程语言**：

- **C/C++** — 嵌入式、仿真器、底层工具的通用语言
- **Python** — 自动化、数据处理、机器学习、EDA 脚本

## 本章内容

- [C / C++](c_cpp.md)
- [Python](python.md)

## 学习建议

!!! tip
    数字 IC 方向：C / Python / Tcl 都要会用。
    模拟 IC 方向：Python（数据处理 + 后处理） + 一点 OceanScript / Tcl 即可。
    嵌入式方向：C 语言要非常扎实。
