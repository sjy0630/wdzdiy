# 论文检索

> 状态：🚧 整理中

## 主流数据库

| 平台 | 用途 |
| --- | --- |
| **IEEE Xplore** | 微电子方向最权威，JSSC / TCAS / VLSI 等顶刊 |
| **ACM Digital Library** | 偏 EDA / 计算机体系结构 |
| **Google Scholar** | 综合搜索 + 引用追踪 |
| **DBLP** | 计算机 / EDA 顶会列表 |
| **Semantic Scholar** | AI 增强语义搜索，找综述方便 |

## 微电子顶级会议 / 期刊

- **JSSC** — IEEE Journal of Solid-State Circuits
- **ISSCC** — Solid-State Circuits Conf.（业界春晚）
- **VLSI Symposium** — 工艺 + 电路
- **DAC / ICCAD** — EDA / 设计自动化
- **MICRO / ISCA / HPCA** — 体系结构
- **IEDM** — 器件物理

## 工具推荐

- **Zotero** — 文献管理与笔记
- **Connected Papers** — 可视化探索相关论文
- **Litmaps** — 文献时间线追踪
