# 英语

> 状态：🚧 整理中

## 为什么重要

- **绝大多数权威教材是英文**（Razavi、Sedra、Sze 等）
- **顶级论文 100% 英文**
- **求职 / 跳槽**：外企、海外读博都需要
- **AI / 工具文档**永远是英文先行

## 训练资源

### 阅读

- 直接读英文原版教材，比看翻译版效率更高
- The Economist / IEEE Spectrum
- Anki + 关键词卡片（专业词汇）

### 听说

- YouTube：MIT OCW / Stanford 公开课
- Razavi 讲课视频（全英文，专业词汇全覆盖）
- 找一个 Language Exchange 伙伴

### 写作

- **《Style: Lessons in Clarity and Grace》** — Joseph Williams
- **《On Writing Well》** — William Zinsser
- 投稿前先用 **Grammarly / DeepL Write** 检查
