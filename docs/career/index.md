# 科研与求职

技术栈之外，**做研究的方法**和**求职的信息差**也是大学阶段需要补的两块短板。

## 本章内容

- [论文检索](literature.md) — IEEE / ACM / Google Scholar 的高效用法
- [英语](english.md) — 文献阅读、技术写作、口语
- [实习与保研](intern.md) — 信息源、时间线、准备策略

## 学习建议

!!! note "什么时候开始关心这些"
    - **大一**：英语、保持好奇心，多看综述
    - **大二**：进实验室，跟一两个学长做项目
    - **大三**：明确方向，开始投实习
    - **大四**：保研 / 考研 / 就业 三选一并冲刺
